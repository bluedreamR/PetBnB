# API Endpoints

## HTML API

### Root

- `GET /` - loads React web app

## JSON API

### Users

- `POST /api/users`
- `PATCH /api/users`

### Session

- `POST /api/session`
- `DELETE /api/session`
- `GET /api/session`

### Pets

- `GET /api/locations/:location_id/pets`
- `GET /api/locations/:location_id/pets/:id`

### Reviews

- `GET /api/locations/:location_id/pets/:pet_id/reviews`

### Locations

- `GET /api/locations`
- `GET /api/location/:id`
